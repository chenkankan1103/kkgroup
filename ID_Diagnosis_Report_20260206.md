### 4. 問題根源

**主要原因**: 昵稱格式變化，而非系統 ID 偏差

Discord 成員的昵稱被更改為包含編號標籤的格式:
- 原始格式: 用戶昵稱
- 新格式: "No.XXXXX 用戶昵稱" 或 "NO.XXXXX 用戶昵稱"

這導致基於昵稱的匹配失敗，但 ID 實際上大多是正確的。

### 5. 影響範圍
- KKcoin 排行榜: ❌ 無法正確映射用戶（昵稱匹配失敗）
- 使用者頭像: ❌ 可能無法加載（ID 不匹配部分情況）
- 受影響用戶: 48/49 個用戶（大部分 ID 正確，但無昵稱映射）
